Headings and Paragraphs

HTML uses <h1> to <h6> tags for headings to show document structure, with <h1> being the most important.

A Subheading (h3)

This is a standard paragraph defined by the <p> tag. You cannot be sure how HTML will be displayed across different screen sizes, so you shouldn't add extra spaces in your code for layout.

Lists

HTML supports both unordered and ordered lists.

Unordered List (Bullet Points)

  • First item in the list
  • Second item, which can contain a link to another page.
  • Third item using a bullet point.

Ordered List (Numbered Items)

  1. Step one: Do something.
  2. Step two: Do something else.
  3. Step three: Finish the task.

Images and Links

The <img> tag is used for images, and it uses attributes like src (source file) and alt (alternative text).

Links are defined using the <a> anchor tag, with the destination specified in the href attribute.
